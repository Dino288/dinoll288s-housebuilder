# DinoLL288s Housebuilder 0.1.0

Initial release for Minecraft 1.20.1 Forge.

- Adds a client-side housebuilder GUI
- Adds quick build support with the `B` key
- Supports customizable 5x5 and 7x7 starter houses
- Supports wall, floor, and roof material selection from the held block
- Supports creative and survival placement using normal player actions

Tested artifact:

- Forge 1.20.1
